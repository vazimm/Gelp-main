Módulo 4:Introdução ao CSS


O que é o CSS:


O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML.
Ele permite que você controle o layout, as cores, as fontes e outros aspectos visuais de uma página web.
O CSS é uma parte fundamental do desenvolvimento web moderno, permitindo que os desenvolvedores criem interfaces atraentes e responsivas. Com o CSS, você pode aplicar estilos a elementos HTML de forma eficiente e reutilizável.
O CSS é uma linguagem de estilo que pode ser aplicada a qualquer tipo de documento HTML, e é amplamente utilizado em conjunto com outras tecnologias web, como JavaScript e frameworks front-end.

Por que usar o CSS:


O CSS permite separar o conteúdo da apresentação. Isso significa que você pode alterar a aparência de um site sem precisar modificar o HTML. Além disso, o CSS oferece uma série de vantagens, como:

  • Facilidade de manutenção
  • Reutilização de estilos
  • Compatibilidade com dispositivos móveis
  • Melhoria no desempenho do site
  • Flexibilidade e controle total sobre o layout

Formas de Aplicação do CSS:


O CSS pode ser aplicado de três maneiras diferentes:

  • Inline CSS: Estilos aplicados diretamente nos elementos HTML usando o atributo "style".
  • Internal CSS: Estilos definidos dentro da tag <style> no cabeçalho do documento HTML.
  • External CSS: Estilos definidos em um arquivo CSS separado, que é vinculado ao documento HTML usando a tag <link>.

CSS Inline:

O CSS inline é aplicado diretamente nos elementos HTML usando o atributo "style". Isso significa que você pode definir estilos específicos para um único elemento sem afetar outros elementos na página.

<p style="color: red;">Texto vermelho</p>

CSS Interno:

O CSS interno é definido dentro da tag <style> no cabeçalho do documento HTML. Isso permite que você aplique estilos a vários elementos na mesma página sem precisar repetir o código.

<style>
    body {
        background-color: lightblue;
    }
</style>

CSS Externo:

O CSS externo é definido em um arquivo CSS separado, que é vinculado ao documento HTML usando a tag <link>. Isso permite que você reutilize os mesmos estilos em várias páginas da web.

<link rel="stylesheet" href="styles.css">

Exemplo de conteúdo do arquivo style.css

body {
    background-color: lightblue;
}
h1 {
    color: white;
    text-align: center;
}
p {
    font-size: 20px;
    color: black;
}

Esse exemplo mostra a estrutura básica de um arquivo de estilo CSS externo.


Sintaxe do CSS:


A sintaxe do CSS é simples e fácil de entender. Cada regra CSS consiste em um seletor e um bloco de declarações. O seletor indica qual elemento HTML será estilizado, enquanto o bloco de declarações contém as propriedades e valores que serão aplicados a esse elemento.

Cada regra CSS segue a estrutura abaixo:

seletor {
propriedade: valor;
}


h1 {
color: blue;
}

Neste exemplo, o seletor é "h1", que se refere a todos os elementos <h1> da página. A propriedade "color" define a cor do texto como azul.


Seletores:


Os seletores são usados para selecionar os elementos HTML que você deseja estilizar. Existem vários tipos de seletores no CSS, incluindo:
Selotor de Tag: Seleciona todos os elementos de uma determinada tag HTML. Por exemplo, h1 irá selecionar todos os elementos <h1> na página.
Seletor de Classe: Seleciona todos os elementos que possuem uma determinada classe. Para usar um seletor de classe, você deve preceder o nome da classe com um ponto (.). Por exemplo, .minha-classe irá selecionar todos os elementos que possuem a classe "minha-classe".
Seletor de ID: Seleciona um elemento específico com um ID exclusivo. Para usar um seletor de ID, você deve preceder o ID com um símbolo de hash (#). Por exemplo, #meu-id irá selecionar o elemento com o ID "meu-id".
Abaixo estão alguns exemplos de seletores CSS:

h1 {
    color: blue;
}
.minha-classe {
    background-color: yellow;
}
#meu-id {
    font-size: 24px;
}

Abaixo está como os seletores são inseridos no HTML:

<h1 class="minha-classe" >Título</h1>
<h3 id = "subtitulo" >Subtítulo</h3>

Dicas Importantes:
1. Use seletores de classe para aplicar estilos a grupos de elementos semelhantes.
2. Use seletores de ID para aplicar estilos a elementos específicos.
3. Evite usar seletores inline, pois eles podem dificultar a manutenção do código.
4. Use seletores de tag para aplicar estilos a todos os elementos de uma determinada tag, mas tenha cuidado para não afetar outros elementos indesejados.
5. Combine seletores para aplicar estilos a elementos que atendem a várias condições. Por exemplo, h1.minha-classe irá selecionar todos os elementos <h1> que possuem a classe "minha-classe".
6. Use seletores descendentes para aplicar estilos a elementos que estão dentro de outros elementos. Por exemplo, div p irá selecionar todos os elementos <p> que estão dentro de um elemento <div>.

Prática:

Agora que você conheceu os conceitos básicos de CSS, coloque em prática criando sua própria página com estilos personalizados!
Você pode usar os exemplos acima como ponto de partida. Lembre-se de experimentar diferentes seletores e propriedades CSS para ver como eles afetam a aparência da sua página. 


                        <--html-->
                        <!DOCTYPE html>
                        <html>
                            <head>
                                <meta charset="UTF-8">
                                <title>CSS Exemplo</title>
                                <link rel="stylesheet" href="estilo.css">
                            </head>
                            <body>
                                <h1 id="titulo-principal">Meu Site com CSS</h1>
                                <p class="texto-destaque">Este parágrafo usa uma classe.</p>
                                <p>Este parágrafo usa estilo padrão.</p>
                            </body>
                        </html>

                        <--style-->
                            /* CSS - estilo.css */
                            #titulo-principal {
                                color: darkred;
                            }

                            .texto-destaque {
                                background-color: yellow;
                                font-style: italic;
                            }

                            p {
                                font-family: Verdana;
                                font-size: 16px;
                            }
