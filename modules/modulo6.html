Módulo 6: Layouts com CSS, Flexbox e Grid


Introdução: Flexbox e Grid

Até agora, aprendemos a estilizar elementos isolados com cores, fontes, bordas e espaçamentos. Mas quando queremos organizar vários elementos juntos — como menus, seções de uma página, colunas ou áreas laterais — precisamos de ferramentas mais poderosas de layout.

É aí que entram o Flexbox e o CSS Grid.

Essas duas tecnologias modernas do CSS nos permitem criar layouts responsivos, centralizar elementos com facilidade, alinhar componentes em linhas ou colunas e controlar a distribuição de espaço na tela — sem depender de gambiarras com margens ou floats.

  • O Flexbox é ideal para alinhar itens em uma dimensão (horizontal ou vertical).
  • O CSS Grid é excelente para estruturar dois eixos ao mesmo tempo (linhas e colunas).

Com esses dois recursos, você terá controle total sobre o posicionamento e o comportamento dos elementos em diferentes tamanhos de tela.

Flexbox - Alinhamento em linha e coluna


O Flexbox (flexible box) permite dispor e alinhar elementos em uma linha ou coluna, adaptando-se facilmente a diferentes tamanhos de tela.
Ativando o Flexbox: Para ativar o Flexbox, adicione a propriedade display: flex; ao elemento CSS. Isso fará com que todos os elementos filhos se comportem como itens flexíveis.


display: flex; Ativa o flexbox
.container {
  display: flex;
}

Alinhamentos úteis:


justify-content: Alinha os itens ao longo do eixo principal (horizontal).


align-items: Alinha os itens ao longo do eixo transversal (vertical).


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

justify-content: center; centraliza os itens horizontalmente.

align-items: center; centraliza os itens verticalmente.


Exemplo prático:Pegue o exemplo abaixo e o execute.


--html--
 <div class="flex-container">
      <div>Item 1</div>
      <div>Item 2</div>
      <div>Item 3</div>
 </div>

--css--
.flex-container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  height: 150px;
  background-color: #f0f0f0;
}
.flex-container div {
  background: #3498db;
  color: white;
  padding: 20px;
}

CSS Grid - Controle total do layout em 2D


O CSS Grid permite criar layouts bidimensionais, oferecendo controle total sobre a disposição dos elementos na página. É ideal para estruturas complexas e responsivas, pois possibilita dividir o espaço em linhas e colunas, formando uma grade flexível e adaptável.
Ativando o Grid: Para ativar o CSS Grid, adicione a propriedade display: grid; ao elemento CSS. Isso fará com que todos os elementos filhos se comportem como itens de grid.


display: grid; ativa o CSS Grid.

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

Exemplo prático:Pegue o exemplo abaixo e o execute.


--html--
<div class="grid-container">
    <div>Área 1</div>
    <div>Área 2</div>
    <div>Área 3</div>
</div>

--css--
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
  background-color: #ecf0f1;
}
.grid-container div {
  background: #2ecc71;
  padding: 20px;
  color: white;
  text-align: center;
}

Complemento: Estruturando Grid com grid-template-columns e grid-template-rows


O CSS Grid permite definir o número de colunas e linhas desejadas, além de especificar o tamanho de cada uma delas. Para isso, utilizamos as propriedades grid-template-columns e grid-template-rows.


grid-template-columns: define o número de colunas e seus tamanhos.


Exemplo 1:


.grid {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;
}

No exemplo acima, a primeira coluna terá 200 pixels de largura, enquanto as outras duas ocuparão o restante do espaço disponível (1fr cada).
Obs: cada valor representa o tamanho de cada coluna.


Exemplo 2:

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

No exemplo acima, o grid terá 3 colunas de tamanhos iguais, ocupando igualmente o espaço disponível.
Obs: o repeat(3, 1fr) indica que queremos repetir a coluna 3 vezes, cada uma ocupando 1 fração do espaço disponível.


grid-template-rows: define o número de linhas e seus tamanhos.


Exemplo 1:


.grid {
  display: grid;
  grid-template-rows: 100px auto 50px;
}

No exemplo acima, a primeira linha terá 100 pixels de altura, a segunda ocupará o espaço restante (auto) e a terceira terá 50 pixels de altura.
Obs: cada valor representa o tamanho de cada linha.


Dica Visual: Se pensarmos em uma tabela invisível, grid-template-columns define quantas colunas terá essa tabela, e grid-template-rows define quantas linhas.


Dicas Rápidas: Conceitos entre Flexbox e Grid

Conceito Flexbox Grid
Layout 1 dimensão (linha ou coluna) 2 dimensões (linha e coluna)
Sintaxe display: flex; display: grid;
Controle Posicionamento flexível Divisão de áreas e tamanhos
Ideal para Menus, cards, alinhamento simples Layouts de páginas inteiras

Valores úteis:


Valor Significado
px Tamanho fixo (ex: 200px)
% Porcentagem em relação ao elemento pai
fr Fração do espaço disponível
auto Adapta-se ao conteúdo do elemento
minmax() Define valor mínimo e máximo para a área
repeat() Atalho para repetir valores (ex: 3 colunas)

Atividade Prática – Layout com Flexbox e Grid


Objetivo:
Criar uma estrutura de página simples, aplicando Flexbox para alinhar um cabeçalho e um rodapé, e CSS Grid para organizar a área de conteúdo em colunas.


Desafio: "Página de Apresentação Pessoal"


Crie uma página HTML com o seguinte layout:

                    +------------------------------+ 

                    |         Cabeçalho            |  ← Flexbox (centralizado) 

                    +------------------------------+ 
 
                    |  Foto |  Sobre mim | Contato |  ← Grid (3 colunas) 
 
                    +------------------------------+ 

                    |         Rodapé               |  ← Flexbox (centralizado) 

                    +------------------------------+

Requisitos técnicos

Estrutura:

Crie dois arquivos: index.html e estilo.css
No HTML:

  • Um elemento <header> para o cabeçalho
  • Uma <section class="conteudo"> com 3 <div>:
    • Foto (pode usar uma <img>)
    • Texto de apresentação
    • Informações de contato
  • Um <footer> com uma frase simples

Estilização:

  • 1. Use Flexbox no <header> e no <footer> para centralizar o conteúdo.
  • 2. Use Grid na .conteudo para criar 3 colunas.
  • 3. Defina uma largura máxima de 900px e centralize a página (margin: auto).
  • 4. Dê um fundo claro ao conteúdo e destaque visual aos blocos (ex: com borda, sombra, padding).
  • Adicione gap entre as colunas do Grid.
  • 5. Faça com que, em telas pequenas, as colunas virem uma abaixo da outra (responsividade com grid-template-columns adaptativo).

Dica:

.conteudo {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

Obs:O auto-fit é usado para ajustar o número de colunas automaticamente, enquanto o minmax(250px, 1fr) garante que cada coluna tenha no mínimo 250px e ocupe o espaço restante disponível.
