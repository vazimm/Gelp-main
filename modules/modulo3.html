Módulo 3: Listas, Tabelas, e Formulários em HTML


Lista em HTML:


Listas Não Ordenadas:

As listas não ordenadas são usadas para agrupar itens que não têm uma ordem específica.
Elas são criadas usando a tag <ul> (lista não ordenada) e cada item da lista é definido com a tag
<li> (item da lista). Veja um exemplo:
 <ul>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
 </ul>

Listas Ordenadas:

As listas ordenadas são usadas para agrupar itens que têm uma ordem específica.
Elas são criadas usando a tag <ol> (lista ordenada) e cada item da lista é definido com a tag
<li> (item da lista). Veja um exemplo:
 <ol>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
 </ol>

Tabelas em HTML:


As tabelas são usadas para organizar dados em linhas e colunas. Elas são criadas usando a tag <table>
(tabela), e cada linha da tabela é definida com a tag <tr> (linha da tabela). As células da tabela são
definidas com a tag <td> (célula da tabela). Veja um exemplo:
 <table border="1">
   <tr>
     <td>Célula 1</td>
     <td>Célula 2</td>
   </tr>
 </table>
  • <table> : Cria a tabela
    Obs:border e um recurso CSS usado para criar a borda da tabela.
  • <tr> : Cria uma linha na tabela.
  • <td> : Cria uma célula na tabela.
  • <th> : Cria um cabeçalho na tabela.

Formulários em HTML:


Os formulários são usados para coletar dados do usuário. Eles são criados usando a tag <form> (formulário),
usamos a tag <label> (rótulo) para descrever o campo de entrada,e cada campo do formulário é definido com
a tag <input> (campo de entrada). Existem vários tipos de campos de entrada:

Sendo os mais comuns:

  • text : Campo de texto simples.
  • email : Campo para inserir um endereço de e-mail.
  • number : Campo para inserir um número.
  • password : Campo para inserir uma senha.
  • radio : Botão de opção (radio button).
  • checkbox : Caixa de seleção (checkbox).
  • submit : Botão de envio do formulário.

Veja um exemplo:
 <form>
   <label for="name">Nome:</label>
   <input type="text" id="name" name="name">
   <label for="email">Email:</label>
   <input type="email" id="email" name="email">
   <button type="submit">Enviar</button>
 </form>

Conexão do formulario com o Back-end

Obs: O foco do curso é em front-end, mas é importante entender como os formulários funcionam com o back-end.
Quando um usuário preenche um formulário e clica no botão de envio, os dados do formulário são enviados para o servidor.
O servidor processa os dados e pode armazená-los em um banco de dados, enviar um e-mail ou realizar outras ações.
O back-end é responsável por lidar com a lógica do servidor, como validação de dados, autenticação de usuários e interação com bancos de dados.
O front-end, por outro lado, é responsável pela interface do usuário e pela apresentação dos dados.
Para conectar o front-end ao back-end, você pode usar tecnologias como AJAX (Asynchronous JavaScript and XML) ou Fetch API para enviar dados do formulário para o servidor sem recarregar a página. Isso permite uma experiência de usuário mais fluida e interativa.

Para tratar os dados do formulário, você pode usar JavaScript ou enviar os dados para um servidor usando uma linguagem de back-end como PHP, Python, etc.
Atráves do atributo action na tag <form>, você pode especificar o URL para onde os dados do formulário devem ser enviados.
Usando o atributo method, você pode especificar o método HTTP a ser usado para enviar os dados do formulário. Os métodos mais comuns são GET e POST.
O método GET envia os dados do formulário como parte da URL, enquanto o método POST envia os dados no corpo da solicitação HTTP.

Veja um exemplo de como usar os atributos action e method:
 <form action="processar_formulario.php" method="POST">
   <label for="name">Nome:</label>
   <input type="text" id="name" name="name">
   <label for="email">Email:</label>
   <input type="email" id="email" name="email">
   <button type="submit">Enviar</button>
 </form>

Validação de Formulários

A validação de formulários é o processo de garantir que os dados inseridos pelo usuário estejam corretos e completos antes de serem enviados para o servidor.
A validação pode ser feita no lado do cliente (front-end) usando JavaScript ou no lado do servidor (back-end) usando uma linguagem de programação como PHP, Python, etc.
A validação no lado do cliente é útil para fornecer feedback imediato ao usuário, enquanto a validação no lado do servidor é importante para garantir a segurança e integridade dos dados.
Existem várias maneiras de validar formulários em HTML, incluindo:

  • Usar atributos HTML5 como required, minlength, maxlength, pattern, etc.
  • Usar JavaScript para validar os dados do formulário antes de enviá-los para o servidor.
  • Usar bibliotecas de validação de formulários, como jQuery Validation ou Parsley.js.
  • Usar frameworks de front-end, como React ou Angular, que possuem recursos de validação de formulários integrados.

Veja um exemplo de validação de formulário usando o atributo required:
 <form>
   <label for="name">Nome:</label>
   <input type="text" id="name" name="name" required>
   <label for="email">Email:</label>
   <input type="email" id="email" name="email" required>
   <button type="submit">Enviar</button>
 </form>

Exemplo de validação de formulário usando HTML5: 
                    <!DOCTYPE html>
                    <html lang="pt-BR">
                    <head>
                        <meta charset="UTF-8">
                        <title>Formulário com Validação</title>
                    </head>
                    <body>
                        <main>
                            <section>
                                <h2>Cadastro</h2>
                                <form>
                                    <!-- Campo Nome -->
                                    <label for="nome">Nome:</label><br>
                                    <input type="text" id="nome" name="nome" required><br><br>

                                    <!-- Campo E-mail -->
                                    <label for="email">E-mail:</label><br>
                                    <input type="email" id="email" name="email" required><br><br>

                                    <!-- Campo Senha -->
                                    <label for="senha">Senha:</label><br>
                                    <input type="password" id="senha" name="senha" minlength="6" required><br><br>

                                    <!-- Campo Idade -->
                                    <label for="idade">Idade:</label><br>
                                    <input type="number" id="idade" name="idade" min="18" max="100" required><br><br>

                                    <!-- Botão de Enviar -->
                                    <input type="submit" value="Enviar">
                                </form>
                            </section>
                        </main>
                    </body>
                    </html>

  • required : Indica que o campo é obrigatório.
  • minlength : Define o número mínimo de caracteres para o campo.
  • maxlength : Define o número máximo de caracteres para o campo.
  • type="email" : Valida o formato do e-mail.
  • type="password" minlength="6" : define o campo como senha e exige no mínimo 6 caracteres.
  • type="radio" name=sexo (mesmo nome indica o grupo) :
  • textarea require : impede envio sem mensagem

Exercício Prático

Agora é a sua vez, Crie um formulário simples em HTML que contenha os seguintes campos:

  • Nome (campo de texto)
  • Email (campo de email)
  • Senha (campo de senha)
  • Sexo (opções de rádio: Masculino, Feminino)
  • Interesses (caixas de seleção: Música, Esportes, Tecnologia)
  • Mensagem (campo de texto grande)
  • Botão de Enviar

Desafio: Adicione validação ao formulário para garantir que todos os campos sejam preenchidos corretamente antes de serem enviados.
Obs: Você pode usar os atributos HTML5 mencionados acima ou escrever seu próprio código JavaScript para validar os dados do formulário.

Opinião do instrutor:

Abaixo se encontra a resolução do exercício, mas não olhe antes de tentar resolver o exercício por conta própria. 


                        <!DOCTYPE html>
                        <html lang="pt-BR">
                        <head>
                            <meta charset="UTF-8">
                            <title>Formulário de Cadastro</title>
                        </head>
                        <body>
                            <h2>Formulário de Cadastro</h2>

                            <form action="processa.php" method="post">
                                
                                <!-- Nome -->
                                <label for="nome">Nome:</label><br>
                                <input type="text" id="nome" name="nome" required><br><br>

                                <!-- Email -->
                                <label for="email">Email:</label><br>
                                <input type="email" id="email" name="email" required><br><br>

                                <!-- Senha -->
                                <label for="senha">Senha (mínimo 6 caracteres):</label><br>
                                <input type="password" id="senha" name="senha" required minlength="6"><br><br>

                                <!-- Sexo -->
                                <label>Sexo:</label><br>
                                <input type="radio" id="masculino" name="sexo" value="Masculino" required>
                                <label for="masculino">Masculino</label><br>
                                <input type="radio" id="feminino" name="sexo" value="Feminino">
                                <label for="feminino">Feminino</label><br><br>

                                <!-- Interesses -->
                                <label>Interesses:</label><br>
                                <input type="checkbox" id="musica" name="interesses[]" value="Música">
                                <label for="musica">Música</label><br>
                                <input type="checkbox" id="esportes" name="interesses[]" value="Esportes">
                                <label for="esportes">Esportes</label><br>
                                <input type="checkbox" id="tecnologia" name="interesses[]" value="Tecnologia">
                                <label for="tecnologia">Tecnologia</label><br><br>

                                <!-- Mensagem -->
                                <label for="mensagem">Mensagem:</label><br>
                                <textarea id="mensagem" name="mensagem" rows="4" cols="50" required></textarea><br><br>

                                <!-- Botão de Enviar -->
                                <input type="submit" value="Enviar">

                            </form>
                        </body>
                        </html>

  • required : Indica que o campo é obrigatório.
  • minlength : Define o número mínimo de caracteres para o campo.
  • maxlength : Define o número máximo de caracteres para o campo.
  • pattern : Define uma expressão regular que o valor do campo deve corresponder.
  • type="radio" : name (define o nome do grupo de opções). Se usar o required junto, obriga o usuário a escolher uma opção (sexo).
    Obs: Volte ao exemplo e analise a entrada do campo sexo do usuário.
  • type="checkbox" : Permite que o usuário selecione várias opções (interesses).
  • textarea require : Impede envio sem mensagem.
