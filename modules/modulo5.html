Módulo 5: Estilização Avançada com CSS


Introdução: Propriedades do CSS


Agora que você já sabe aplicar CSS e usar seletores básicos, é hora de aprender como deixar suas páginas mais bonitas com estilização visual mais completa: cores, fundos, bordas, espaçamentos e tamanhos. Essas propriedades são a base para estruturar visualmente seus elementos HTML.

Cores e Fundos:

As cores podem ser definidas de várias maneiras: por nome, código hexadecimal ou RGB. Você pode aplicar cores a textos, fundos e bordas. Além disso, é possível usar imagens como fundo.


color: define a cor do texto

color: blue;


background-color: define a cor de fundo

background-color: lightblue;


background-image: define uma imagem de fundo

background-image: url('imagem.jpg');

Você também pode usar:

  • nomes de cores: red, blue, green, yellow, pink, black.
    Obs: o CSS possui 140 cores nomeadas oficialmente.
  • códigos hexadecimais: #ff0000
  • RGB: rgb(255, 0, 0)
    Obs: opacidade pode ser definida com rgba(255, 0, 0, 0.5)
    o último número define a opacidade (0 a 1).

Bordas:

As bordas podem ser aplicadas a qualquer elemento. Você pode definir a largura, estilo e cor da borda.
Os estilos mais comuns são: solid, dashed e dotted.

  • Solid: linha sólida
  • Dashed: linha tracejada
  • Dotted: linha pontilhada


border: define a borda de um elemento

border: 2px solid red;

Acima, a borda é de 2 pixels de largura, do tipo sólida e da cor vermelha. chamamos a borda de "border" e ela é composta por três partes:
definimos a sua largura, o estilo e a cor. Conforme demonstrado acima. Podemos fazer isso de forma separada também em um arquivo CSS,
como no exemplo abaixo:

border-width: 2px;
border-style: solid;
border-color: red;


border-radius: define o arredondamento das bordas

border-radius: 10px;

Tamanhos (width e height):

As propriedades width e height definem a largura e altura de um elemento. Você pode usar pixels, porcentagens ou outras unidades de medida para especificar o tamanhos de seus elementos.

width: define a largura de um elemento

width: 200px;


height: define a altura de um elemento

height: 100px;


max-width: define a largura máxima de um elemento

max-width: 100%;


min-width: define a largura mínima de um elemento

min-width: 50px;


max-height: define a altura máxima de um elemento

max-height: 100%;


min-height: define a altura mínima de um elemento

min-height: 50px;

podemos dimensionar o tamanho de elementos como textos, tabelas e bordas

Este texto está com tamanho de 24px.

Espaçamento (padding e margin):

O padding é o espaço interno de um elemento, enquanto a margin é o espaço externo. Você pode usar essas propriedades para criar espaçamentos entre elementos e ajustar o layout da sua página.

padding: define o espaçamento interno de um elemento

padding: 20px;


margin: define o espaçamento externo de um elemento

margin: 10px;

Podemos definir as dimensões laterais no padding e margin em uma unica propriedade da seguinte forma:

padding: 10px 15px 20px 25px;
margin: 20px 15px 20px 15px;

Dessa forma estamos dimensionando em uma unica propriedade apartir da seguinte ordem: Superior, Direito, Inferior, Esquerdo (Seguindo o sentido horário).

Outras formas de dimensonamentos:


Podemos fazer de forma mais especifica o dimensionamento do padding e margin, através das seguintes propriedades

  • margin-top: define o espaçamento externo superior de um elemento
  • margin-bottom: define o espaçamento externo inferior de um elemento
  • margin-left: define o espaçamento externo esquerdo de um elemento
  • margin-right: define o espaçamento externo direito de um elemento
  • padding-top: define o espaçamento interno superior de um elemento
  • padding-bottom: define o espaçamento interno inferior de um elemento
  • padding-left: define o espaçamento interno esquerdo de um elemento
  • padding-right: define o espaçamento interno direito de um elemento

Alinhamentos:

O alinhamento de texto pode ser feito com as propriedades text-align, vertical-align e line-height. Você pode alinhar o texto à esquerda, direita ou centralizado.

text-align: define o alinhamento do texto

text-align: center;

Obs: Acima o texto e alinhado ao centro, utilizando o valor center. Mas também podem ser, usados os valores: left (esquerda) ou right (direita).


vertical-align: define o alinhamento vertical do texto

vertical-align: middle;

Obs: A propriedade vertical-align é usada para alinhar elementos inline (como textos ou imagens) verticalmente em relação à linha base do texto. No exemplo acima, o valor middle centraliza o conteúdo verticalmente na linha. Também é comum usar top, bottom ou baseline.


line-height: define a altura da linha do texto

line-height: 1.5;

Obs: A propriedade line-height define a altura da linha, controlando o espaçamento vertical entre as linhas de texto. No exemplo acima, o valor 1.5 aumenta o espaço, facilitando a leitura. Pode ser definido com número, unidade (px, em) ou porcentagem.


text-decoration: define a decoração do texto

text-decoration: underline;

Obs: A propriedade text-decoration aplica decorações ao texto. No exemplo acima, o valor underline sublinha o texto. Outros valores possíveis incluem overline (linha acima), line-through (risco no meio) e none (sem decoração).


Atividade Prática

Objetivo: Criar uma caixa de conteúdo com: Um título; Um parágrafo; Um botão estilizado; Aplicar borda, fundo, padding, margem e alinhamento.

Passo 1: Crie um arquivo HTML e um arquivo CSS separados.

Passo 2: No arquivo HTML, crie uma estrutura básica com um título, um parágrafo e um botão.

Passo 3: No arquivo CSS, aplique as seguintes propriedades:

  • Defina uma borda de 2 pixels, sólida e azul para a caixa.
  • Defina um fundo claro (ex: #f0f0f0) para a caixa.
  • Adicione um padding de 20 pixels ao redor do conteúdo da caixa.
  • Adicione uma margem de 30 pixels ao redor da caixa.
  • Alinhe o texto do título ao centro e defina uma cor diferente (ex: #3498db).

Passo 4: No botão, aplique um fundo azul, texto branco e arredondamento de bordas (border-radius) de 5 pixels.

Passo 5: Teste o resultado no navegador.

Passo 6: Faça ajustes conforme necessário para melhorar a aparência.

Dica: Caso tenha duvida sobre como criar a caixa você pode utilizar uma div. 


                        <!-- HTML(index.html) -->

                        <!DOCTYPE html>
                        <html lang="pt-br">
                        <head>
                            <meta charset="UTF-8">
                            <title>Caixa Estilizada</title>
                            <link rel="stylesheet" href="estilo.css">
                        </head>
                        <body>
                            <div class="caixa">
                                <h2>Bem-vindo ao Módulo 5</h2>
                                <p>Esta é uma caixa com estilos aplicados via CSS externo.</p>
                                <button>Explorar</button>
                            </div>
                        </body>
                        </html>

                        <!-- CSS (estilo.css) -->
                        body {
                            font-family: Arial, sans-serif;
                            background-color: #e6f2ff;
                        }

                        .caixa {
                            border: 2px solid #007BFF;
                            background-color: #f0f0f0;
                            padding: 20px;
                            margin: 30px auto;
                            width: 300px;
                            text-align: center;
                            border-radius: 8px;
                        }

                        .caixa h2 {
                            color: #3498db;
                            margin-bottom: 10px;
                        }

                        .caixa p {
                            color: #333;
                            margin-bottom: 20px;
                        }

                        .caixa button {
                            background-color: #007BFF;
                            color: #fff;
                            padding: 10px 20px;
                            border: none;
                            border-radius: 5px;
                            cursor: pointer;
                        }

                        .caixa button:hover {
                            background-color: #0056b3;
                        }
