Módulo 2: Estrutura Básica de um Documento HTML


Todo documento HTML possui uma estrutura mínima que define o conteúdo e sua organização. Essa estrutura básica é composta por elementos que indicam o início e o fim do documento, o cabeçalho, o corpo e outras informações essenciais. Abaixo, um exemplo de estrutura básica:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
  </head>
  <body>
    <h1>Bem-vindo ao Curso de Desenvolvimento Web!</h1>
    <p>Este é um exemplo básico de documento HTML.</p>
    <a href="https://www.exemplo.com">Clique aqui para visitar um site exemplo</a>
    <img src="imagem-exemplo.jpg" alt="Descrição da imagem">
  </body>
</html>

Explicação dos principais elementos:


  • <!DOCTYPE html>: Declaração que informa ao navegador que o documento é um HTML5.
  • <html>: Elemento raiz que envolve todo o conteúdo da página.
  • <head>: Contém informações sobre o documento, como título e metadados.
  • <meta charset="UTF-8">: Define a codificação de caracteres do documento.
  • <title>: Define o título da página, exibido na aba do navegador.
  • <body>: Contém o conteúdo visível da página, como textos, imagens e links.

Principais Tags Utilizadas


Cabeçalhos (<h1> a <h6>) :

Os cabeçalhos são usados para definir títulos e subtítulos em diferentes níveis.
Existem seis níveis de cabeçalhos, do <h1> (mais importante) ao <h6> (menos importante).


<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Subtítulo Secundário</h3>

Parágrafos (<p>):

Os parágrafos são usados para agrupar texto em blocos. Eles ajudam a organizar o conteúdo e a torná-lo mais legível.


<p>Este é um parágrafo de exemplo.</p>

Links (<a>):

Os links são usados para criar hyperlinks que direcionam o usuário para outras páginas ou sites. O atributo href define o destino do link.


<a href="https://www.exemplo.com">Clique aqui</a>

Imagens (<img>):

As imagens são inseridas na página usando a tag <img>. O atributo src define o caminho da imagem, enquanto o atributo alt fornece uma descrição alternativa.


<img src="imagem-exemplo.jpg" alt="Descrição da imagem">

Tags Semânticas


Header:

A tag <header> é usada para definir o cabeçalho de uma página ou seção. Geralmente contém o logo e a navegação principal (Barra de navegação).


<header>
  <nav>
    <img src="imagem-exemplo.jpg" alt="Descrição da imagem">
    <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#sobre">Sobre</a></li>
      <li><a href="#contato">Contato</a></li>
    </ul>
  </nav>
</header>

Main:

A tag <main> é usada para definir o conteúdo principal de uma página. Ela destaca a parte mais relevante para o usuário.


<main>
  <h1>Bem-vindo ao Curso de Desenvolvimento Web!</h1>
  <p>Este é um exemplo básico de documento HTML.</p>
</main>

Nav:

A tag <nav> é usada para definir uma seção de navegação em uma página. Geralmente contém links para outras páginas ou seções do site.


<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#sobre">Sobre</a></li>
    <li><a href="#contato">Contato</a></li>
  </ul>
</nav>

Section:

A tag <section> é usada para definir uma seção independente de conteúdo dentro de um documento. É útil para agrupar conteúdo relacionado.


<section>
  <h2>Título da Seção</h2>
  <p>Conteúdo da seção.</p>
</section>

Footer:

A tag <footer> é usada para definir o rodapé de uma página ou seção. Geralmente contém informações de copyright, links de contato e outras informações relevantes.


<footer>
  <p>© 2023 Curso Iniciando na Programação. Todos os direitos reservados.</p>
  <p>Desenvolvido por [Nome Do Grupo]</p>
</footer>

Boas Pratícas para um Código HTML limpo


Identação e organização:

Mantenha seu código HTML bem estruturado e organizado, utilizando recuos consistentes para facilitar a leitura e a manutenção. Aplique espaços ou tabulações para que cada elemento aninhado fique devidamente recuado em relação ao seu contêiner pai.
Além disso, agrupe conteúdos relacionados usando tags semânticas, como <section>, <article> e <aside>. Para reforçar a clareza e a lógica da estrutura do documento.



Comentários:

Utilize comentários para explicar partes do código que possam ser complexas ou que exijam uma explicação adicional. Isso ajuda outros desenvolvedores (ou você mesmo no futuro) a entenderem rapidamente o propósito de cada seção do código.


<!-- Este é um comentário em HTML -->
<p>Este é um parágrafo de exemplo.</p>

Acessibilidade:

Sempre que possível, adicione atributos alt às imagens para descrever seu conteúdo. Isso é importante para a acessibilidade,
pois permite que leitores de tela interpretem o conteúdo da imagem para usuários com deficiência visual.



Considerações Sobre SEO


Uma boa estruturação com tags semânticas não é apenas uma questão de organização visual, mas também de otimização para mecanismos de busca (SEO). Ao utilizar corretamente as tags, você facilita que buscadores entendam o conteúdo e a hierarquia de informações, o que pode melhorar a posição do seu site nos resultados de pesquisa.


Exemplo Prático:


A seguir, veja um exemplo que integra a estrutura básica de um documento HTML com as tags semânticas e boas práticas discutidas: 

                    
                        <!DOCTYPE html>
                        <html>
                            <head>
                                <meta charset="UTF-8">
                                <title>Exemplo de Site Semântico</title>

                                    <section id="home">

Atividades Práticas:


Atividade 1: Criação da Estrutura Básica de um Documento HTML.

Objetivo: Criar um arquivo HTML que contenha a estrutura mínima exigida.



Instruções:

1. Abra um editor de texto (como Visual Studio Code, Sublime Text ou até mesmo o Bloco de Notas).
2. Crie um novo arquivo e salve-o com a extensão .html (por exemplo, meu_primeiro_site.html).
Obs: Geralmente o nome do primeiro arquivo HTML é index.html, mas você pode nomear como preferir.
3. Insira a estrutura a básica abaixo: 

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <meta charset="UTF-8">
                        <title>Minha Primeira Página</title>
                    </head>
                    <body>
                        <!-- Conteúdo vai aqui -->
                    </body>
                    </html>

4. Salve o arquivo e abra-o em um navegador (como Chrome, Firefox ou Edge) para visualizar sua página.
Conselho: Se você estiver usando o Visual Studio Code, você pode instalar a extensão "Live Server"
para visualizar suas alterações em tempo real.


Atividade 2: Adicionando Conteúdo com Cabeçalhos e Parágrafos

Objetivo: Inserir diferentes niveis de cabeçalhos e parágrafos para praticar a hierarquia de conteúdo.


Instruções:

1. Dentro da tag <body>, do seu arquivo.html, adicione os seguintes elementos 

                
                    <h1>Título Principal</h1>
                    <h2>Subtítulo</h2>
                    <p>Este é um parágrafo de exemplo.</p>
                    <p>Outro parágrafo com mais informações.</p>
                    <h3>Subtítulo Secundário</h3>
                    <p>Mais informações sobre o assunto.</p>
                    <h4>Subtítulo Terciário</h4>
                    <p>Detalhes adicionais.</p>
                    <h5>Subtítulo Quaternário</h5>
                    <p>Informações complementares.</p>
                    <h6>Subtítulo Quinário</h6>
                    <p>Informações finais.</p>

2. Salve o arquivo e atualize a página no navegador para ver as alterações.
Conselho: Use diferentes níveis de cabeçalhos para organizar o conteúdo de forma hierárquica.
Obs: Se estiver usando o "live-server" ao salvar o codigo, a página será atualizada automaticamente.
3. Para praticar um pouco de formatação: Use a tag <br> Para aplicar espaços,
insira após um trecho do texto ou tag html que você queira que salte uma linha.


Atividade 3: Inserção de Links e Imagens

Objetivo: Criar um links navegável e inserir uma imagem à sua página HTML.

Instruções:

1. Inserindo um link:
Dentro da tag <body>, adicione o seguinte código para criar um link: 

                
                    <a href="https://www.exemplo.com">Clique aqui para visitar um site exemplo</a>

Obs: O atributo href deve conter o URL do site que você deseja vincular.
Obs: O atributo: target="_blank" faz com que o link seja aberto em uma nova aba do navegador.
2. Inserindo uma imagem:
Dentro da tag <body>, adicione o seguinte código para inserir uma imagem:
Obs: O atributo src deve conter o caminho da imagem que você deseja exibir. O caminho pode ser local ou um link. 

                
                    <img src="imagem-exemplo.jpg" alt="Descrição da imagem">

3. Você pode alterar o tamanho da imagem adicionando os atributos width e height, como no exemplo abaixo: 


                
                    <img src="imagem-exemplo.jpg" alt="Descrição da imagem" width="300" height="200">

4. Salve o arquivo e atualize a página no navegador para ver as alterações.
conselho: Use imagens relevantes e links úteis para enriquecer o conteúdo da sua página.
Obs: Se estiver usando o "live-server" ao salvar o código, a página será atualizada automaticamente.
