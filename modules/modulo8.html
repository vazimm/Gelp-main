Módulo 8: Manipulação do DOM (Document Object Model)


Objetivo do Módulo

Entender como acessar, modificar e interagir com elementos HTML usando JavaScript por meio do DOM, tornando as páginas mais dinâmicas e responsivas a eventos do usuário.


O que é DOM?


O DOM (Document Object Model) é uma interface de programação que representa a estrutura de um documento HTML ou XML como uma árvore de objetos. Cada elemento, atributo e texto no documento é representado como um nó na árvore, permitindo que linguagens de programação, como JavaScript, acessem e manipulem esses elementos de forma dinâmica.

O DOM é fundamental para a criação de páginas web interativas, pois permite que os desenvolvedores acessem e modifiquem, em tempo real, o conteúdo, a estrutura e o estilo de um documento HTML. Isso possibilita a criação de efeitos visuais, validação de formulários, manipulação de eventos, entre outras funcionalidades.

Essa estrutura é hierárquica, com o elemento raiz sendo o objeto document. Cada elemento HTML é representado como um nó filho desse objeto, e esses nós podem conter outros nós filhos, formando a estrutura em árvore. Por exemplo, um elemento <div> pode conter vários elementos <p>, <img>, etc., que são considerados nós filhos do <div>.

O DOM é uma interface independente de linguagem, ou seja, pode ser acessado e manipulado por diferentes linguagens de programação. No entanto, o JavaScript é a linguagem mais comum para interagir com o DOM em páginas web. Com ele, os desenvolvedores podem adicionar, remover ou modificar elementos HTML, alterar estilos CSS, responder a eventos do usuário (como cliques e teclas pressionadas) e muito mais.

Como funciona o DOM?


O DOM funciona como uma interface entre o documento HTML e o código JavaScript. Quando um navegador carrega uma página web, ele cria uma representação em memória do documento HTML, que é a árvore DOM. Essa árvore é composta por nós, onde cada nó representa um elemento, atributo ou texto no documento.

O objeto document é fornecido automaticamente pelo navegador e representa a página web carregada. A partir dele, é possível acessar todos os elementos HTML da página. Por exemplo, ao usar document.getElementById("meuElemento"), estamos dizendo ao JavaScript para procurar dentro desse objeto chamado document um elemento com o ID correspondente.

O JavaScript pode acessar e manipular essa árvore usando métodos e propriedades fornecidos pela API do DOM. Por exemplo, você pode usar métodos como getElementById, querySelector, createElement, appendChild, entre outros, para selecionar, criar, adicionar ou remover elementos na árvore DOM.

Além disso, o DOM também permite que você adicione ouvintes de eventos a elementos HTML. Isso significa que você pode reagir a ações do usuário, como cliques, pressionamentos de tecla e movimentos do mouse. Quando um evento ocorre, você pode executar uma função JavaScript para manipular o DOM de acordo com a ação do usuário.

O DOM é dinâmico, o que significa que você pode modificar a estrutura e o conteúdo da página em tempo real. Por exemplo, você pode adicionar novos elementos, alterar o texto de um elemento existente, mudar estilos CSS ou até mesmo remover elementos da página. Essas alterações são refletidas imediatamente na interface do usuário, proporcionando uma experiência interativa e responsiva.

Em resumo, o DOM funciona como uma ponte entre o documento HTML e o código JavaScript, permitindo que os desenvolvedores acessem, manipulem e respondam a eventos em tempo real. Essa capacidade de interação dinâmica é fundamental para criar páginas web modernas e envolventes.


Selecionando elementos no DOM


Para selecionar elementos no DOM, você pode usar vários métodos disponíveis na API do DOM. Aqui estão alguns dos métodos mais comuns:


  • getElementById(id): Seleciona um elemento com o ID especificado.
  • getElementsByClassName(className): Seleciona todos os elementos com a classe especificada.
  • getElementsByTagName(tagName): Seleciona todos os elementos com a tag especificada.
  • querySelector(selector): Seleciona o primeiro elemento que corresponde ao seletor CSS especificado.
  • querySelectorAll(selector): Seleciona todos os elementos que correspondem ao seletor CSS especificado.

Aqui estão alguns exemplos de como usar esses métodos:


Exemplo 1:

const elemento = document.getElementById("meuElemento");
console.log(elemento);

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e armazenando-o na variável elemento. Em seguida, estamos exibindo o elemento no console.


Exemplo 2:

const elementos = document.getElementsByClassName("minhaClasse");
console.log(elementos);

Neste exemplo, estamos selecionando todos os elementos com a classe "minhaClasse" e armazenando-os na variável elementos. Em seguida, estamos exibindo os elementos no console. Observe que getElementsByClassName retorna uma coleção de elementos, então você pode iterar sobre eles usando um loop.


Exemplo 3:

const primeiroElemento = document.querySelector(".minhaClasse");
console.log(primeiroElemento);

Neste exemplo, estamos selecionando o primeiro elemento que corresponde ao seletor CSS ".minhaClasse" e armazenando-o na variável primeiroElemento. Em seguida, estamos exibindo o elemento no console. O método querySelector é muito útil quando você deseja selecionar um único elemento com base em um seletor CSS.


Exemplo 4:

const elementos = document.querySelectorAll("p");
elementos.forEach((elemento) => {
console.log(elemento); });

Neste exemplo, estamos selecionando todos os elementos <p> na página e armazenando-os na variável elementos. Em seguida, usamos o método forEach para iterar sobre cada elemento e exibi-los no console.
Esses métodos de seleção são essenciais para acessar e manipular elementos no DOM. Você pode usá-los para adicionar ouvintes de eventos, alterar estilos, modificar o conteúdo e muito mais. A escolha do método depende do que você deseja selecionar e como deseja interagir com os elementos.


Manipulando elementos no DOM


Depois de selecionar elementos no DOM, você pode manipulá-los de várias maneiras. Aqui estão algumas das operações mais comuns:

  • innerHTML: Altera o conteúdo HTML interno de um elemento.
  • textContent: Altera o conteúdo de texto de um elemento.
  • setAttribute(attr, value): Define um atributo para um elemento.
  • classList.add(className): Adiciona uma classe a um elemento.
  • classList.remove(className): Remove uma classe de um elemento.

Aqui estão alguns exemplos de como usar essas operações:


Exemplo 1:

const elemento = document.getElementById("meuElemento");
elemento.innerHTML = "Novo conteúdo HTML";

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e alterando seu conteúdo HTML interno para "Novo conteúdo HTML".


Exemplo 2:

const elemento = document.getElementById("meuElemento");
elemento.textContent = "Novo conteúdo de texto";

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e alterando seu conteúdo de texto para "Novo conteúdo de texto". Isso é útil quando você deseja alterar apenas o texto exibido, sem afetar o HTML interno.


Exemplo 3:

const elemento = document.getElementById("meuElemento");
elemento.setAttribute("data-atributo", "valor");

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e definindo um atributo personalizado chamado "data-atributo" com o valor "valor". Isso é útil para armazenar informações adicionais em elementos HTML.


Exemplo 4:

const elemento = document.getElementById("meuElemento");
elemento.classList.add("novaClasse");

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e adicionando uma nova classe chamada "novaClasse" a ele. Isso é útil para aplicar estilos CSS adicionais a um elemento.


Exemplo 5:

const elemento = document.getElementById("meuElemento");
elemento.classList.remove("classeExistente");

Neste exemplo, estamos selecionando um elemento com o ID "meuElemento" e removendo uma classe existente chamada "classeExistente" dele. Isso é útil para remover estilos CSS aplicados anteriormente.


Essas operações de manipulação de elementos são essenciais para criar páginas web dinâmicas e interativas. Você pode usá-las para alterar o conteúdo, o estilo e a estrutura dos elementos em resposta a ações do usuário ou eventos na página.

Exemplo Prático


Vamos criar um exemplo prático onde adicionamos um novo elemento à página e alteramos seu conteúdo. Aqui está o código:


const novoElemento = document.createElement("p");
novoElemento.textContent = "Este é um novo parágrafo.";
document.body.appendChild(novoElemento);

Neste exemplo, estamos criando um novo elemento <p> usando o método createElement. Em seguida, definimos seu conteúdo de texto e o adicionamos ao final do corpo do documento usando o método appendChild.
Isso adicionará um novo parágrafo à página com o texto "Este é um novo parágrafo." Você pode experimentar esse código em seu próprio projeto para ver como funciona a criação e adição de novos elementos ao DOM.


Exemplo prático 2


<p id="mensagem">Mensagem original</p>
<button onclick="alterarMensagem()">Clique</button>
<script>
function alterarMensagem() {
const elemento = document.querySelector("#mensagem");
elemento.innerText = "Você clicou no botão!";
}
</script>

Neste exemplo, temos um parágrafo com o ID "mensagem" e um botão. Quando o botão é clicado, a função alterarMensagem é chamada, selecionando o elemento com o ID "mensagem" e alterando seu conteúdo de texto para "Você clicou no botão!".
Isso demonstra como você pode usar o DOM para interagir com elementos HTML e responder a eventos do usuário.
